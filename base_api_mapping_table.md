# UMD Base API Layer Mapping

## Chip / LocalChip / RemoteChip → TTDevice

| **API** | **Parameters / Return** | **UMD Workload** | **UMD Base Layer** | **Description** |
| --- | --- | --- | --- | --- |
| constructor() | **Parameters:** <br> - chip_info: `ChipInfo` — device identification metadata <br> - soc_descriptor: `SocDescriptor` — SOC configuration <br> **Returns:** <br> - N/A | 1. Set power state to high (`TTDevice::set_power_state(true)`) <br> 2. Wait for chip to be ready: <br> &nbsp;&nbsp;• eth core training (`TTDevice::wait_eth_core_training()`) <br> &nbsp;&nbsp;• dram core training (`TTDevice::wait_dram_channel_training()`) <br> 3. | destructor() | **Parameters:** <br> - none <br> **Returns:** <br> - N/A | 1. Set power state to low (`TTDevice::set_power_state(false)`) <br> 2. Manual RAII cleanup| **`TTDevice::set_power_state()`** | Hardware power-down. Puts tensix/MRISC PHYs into reset. |
| start_device() | **Parameters:** <br> - none <br> **Returns:** <br> - `void`| 1. Acquire mutex (`acquire_mutex(CHIP_IN_USE)`) <br> 2. | create() | **Parameters:** <br> - physical_device_id: `int` — device ordinal <br> - sdesc_path: `std::string` — SOC descriptor YAML path (default: `""`, auto-detect) <br> - num_host_mem_channels: `int` — number of host memory channels (default: `0`) <br> - device_type: `IODeviceType` — transport type (default: `PCIe`) <br> **Returns:** <br> - `std::unique_ptr<LocalChip>` | 1.
